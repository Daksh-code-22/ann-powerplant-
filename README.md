# ANN Regression — Power Plant Energy Output Prediction

## Overview
Predicts electrical energy output (PE) of a power plant 
using an Artificial Neural Network built with PyTorch.

## Dataset
- 9,568 data points, 4 features
- Features: Temperature, Exhaust Vacuum, Pressure, Humidity
- Target: Net hourly electrical energy output (PE) in MW

## Model Architecture
- Input layer → 4 features
- Hidden layer 1 → 6 neurons, ReLU
- Hidden layer 2 → 6 neurons, ReLU  
- Output layer → 1 neuron (regression)
